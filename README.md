Hafta 1 — Proje Kurulumu

React Native ve ASP.NET Core projeleri oluşturuldu GitHub reposu açıldı Temel App.tsx ve Program.cs dosyaları yazıldı

Hafta 2 — Veritabanı & Auth API

MSSQL bağlantısı ve Users tablosu oluşturuldu api/auth/register ve api/auth/login endpoint'leri yazıldı Şifre BCrypt ile hashlenir biçimde ve JWT token üretiliyor

Hafta 3 — Login & Kayıt Ekranları

Navigasyon altyapısı kuruldu LoginScreen ve RegisterScreen tamamlandı Başarılı girişte JWT token bellekte saklanıp ana sayfaya yönlendirme yapıldı

Hafta 4 — Kitap İlan Yönetimi

Books tablosu oluşturuldu GET /api/books, GET /api/books/my, POST /api/books, DELETE /api/books/{id} endpoint'leri yazıldı HomeScreen (ilan listesi) ve MyBooksScreen (ilan ekle/sil) tamamlandı

Hafta 5 — Arama, Filtreleme & Kitap Detayı

GET /api/books/search endpoint'i eklendi HomeScreen'e metin arama kutusu ve kategori chip filtreleri eklendi BookDetailScreen tamamlandı; kitap detayları ve ilan sahibi bilgisi gösteriliyor
